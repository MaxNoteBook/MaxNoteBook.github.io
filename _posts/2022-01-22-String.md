# Slicing



- 필요한 만큼만 짤라서 사용



```python
jumin = "990120-1234567"

print("성별 : " + jumin[7])
print("연 : " + jumin[0:2]) # 0 ~ 1
print("월 : " + jumin[2:4])
print("일 : " + jumin[4:6])

print("생년월일 : " + jumin[:6]) # 처음부터 6 직전까지
print("뒤 7자리 : " + jumin[7:]) # 7부터 끝까지
print("뒤 7자리 (뒤에부터) : " + jumin[-7:]) # 맨 뒤에서 7번째부터 끝까지
```

<pre>
성별 : 1
연 : 99
월 : 01
일 : 20
생년월일 : 990120
뒤 7자리 : 1234567
뒤 7자리 (뒤에부터) : 1234567
</pre>


나는 빨간색과 파란색을 좋아해요.
</pre>

# 탈출 문자

- \" \' : 문장 내에서 따옴표

- \\ : 문장 내에서 \

- \r : 커서를 맨 앞으로 이동

- \b : 백스페이스 (한 글자 삭제)

- \t : 탭



```python
print("백문이 불여일견 \n백견이 불여일타")
```

<pre>
백문이 불여일견 
백견이 불여일타
</pre>

```python
# 이것은 "MaxNoteBook" 입니다.
print("이것은 \"MaxNoteBook\" 입니다.")
```

<pre>
이것은 "MaxNoteBook" 입니다.
</pre>

